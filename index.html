iFridge

C, Python, Android
iFridge was a project done in the Hack & Hackathon 2017. It has a multi-item detection system that employs a Convolutional Neural Network to identify the items in the fridge. The dataset was collected and amplified manually through cropping and rotating the photos taken of the groceries in the fridge. The list of items detected is sent over to the Android mobile application, allowing the user to know what is inside their fridge at any point of time. The application also has the function to suggest possible recipes given the existing items in the fridge.

Facial Recognition

Oct - Nov '16
Python
The facial recognition model was created in Python and is able to ‘learn’ the faces of people through the online dataset Labeled Faces in the Wild. When given a new picture of a person in the same dataset, the model can recognize him/her, i.e. by putting a label to the person’s face. Principal Component Analysis (PCA) was employed to extract the key features of each image. Using the reduced images size, we tested various Machine Learning techniques such as Support Vector Machines, Neural Network, Decision Trees, while using 10-fold Cross Validation to check the accuracy rate.
